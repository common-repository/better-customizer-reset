The easiest way to inspect and delete customizer data (theme mods) saved by WordPress themes.


== Description ==

<strong>Inspect and delete Customizer values saved by WordPress themes.</strong>

[Better Customizer Reset](https://www.ilovewp.com/better-customizer-reset/?utm_source=wporg&utm_campaign=readme&utm_medium=link) allows you to view and delete theme settings and customizer settings added as theme_mods.

== Installation ==
Starting with Better Customizer Reset consists of just two steps: installing and activating the plugin.

= Install Better Customizer Reset from within WordPress =

1. Visit the plugins page within your dashboard and select "Add New";
1. Search for and install "Better Customizer Reset";
1. Activate Better Customizer Reset from your Plugins page.

= Install Better Customizer Reset via FTP =

1. Upload the "better-customizer-reset" folder to the /wp-content/plugins/ directory;
1. Activate the Better Customizer Reset plugin through the 'Plugins' menu in WordPress.

= After Activation =

1. You should see a new "Better Customizer Reset" page in the Tools section.

== Frequently Asked Questions ==

= Will this plugin slow down my website? =

No. Better Customizer Reset does not affect the front-end of your website.

= Will this plugin work with my theme? =

Yes, absolutely! This plugin can delete theme mods added by any theme, as long as that theme saves its settings as theme_mods and not as options.
